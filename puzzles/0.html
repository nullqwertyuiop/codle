This is a demo problem before Codle is officially released.

A + B Problem

You are given two integers. You need to compute the sum of them.

Input Format

The input contains two integers a and b (1 ⩽ a, b ⩽ 109) separated by a space.

Output Format

The output contains a single integer: the sum of a and b.

Sample Input

  2 3

Sample Output

  5
